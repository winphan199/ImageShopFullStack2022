# ImageShopFullStack2022

Fullstack webpage used for showing and uploading images. <br/>
Demo: https://main.dlc85sm9x9086.amplifyapp.com/

## Installation

```
git clone <repo_link>
cd <repo>
npm install
```

## Run the program

```
npm start
```

## Technologies

Frontend:
- ReactJS
- Masonary layout
Backend:
- Firebase Storage
- Firebase Authentication
- Firebase Database
- Firestore Database
